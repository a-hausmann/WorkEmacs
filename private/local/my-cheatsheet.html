I use Emacs in Evil mode as I was a Vimmer for around 20 years. I attempt to put most daily stuff into the General/Hydra menu structure, using SPC as "leader" key. But that doesn't work for everything, hence this cheatsheet.

Tip to add environment variables in the locked-down Windows 10 laptop: click the search icon next to Start, type "environment" in the search box and select "Edit environment variables for your account". This will pop up the familiar window to edit the personal environment, possible without having administrator privileges.

Start Calc n/a C-x * calc-dispatch Type "c" or "*" to start/stop RPN Calc; "X" to exit.
Type "p", then "P" to push result into other window at point.
The "p" mappings may be evil-mode remappings, but is effective to place result without leaving Calc Buffer. Remember to place point at desired location before using "p" keys.

Multiple cursors (evil-mc) n/a C-c m m Function: evil-mc-mode
Key chord Action
C-c m m will toggle evil-mc-mode on and off
C-n/C-p mark next/previous occurrance as part of cursors
M-n/M-p cycle through cursors
C-t skip next occurrance of word
g r n/g r p (in normal mode) to skip next or previous occurrance ("g r n" is equivalent to "C-t")
g r u undo last cursor
g r q quit all cursors
